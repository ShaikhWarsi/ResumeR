# Code of Conduct

## Our Pledge
We, as contributors and maintainers of **ResumeR**, pledge to foster an open, welcoming, inclusive, and harassment-free environment for everyone who participates in this project.

ResumeR is built by and for a diverse global community, and we are committed to making collaboration respectful, productive, and enjoyable for all, regardless of background, experience level, or identity.

---

## Our Standards

### Positive behavior includes:
- Using respectful, inclusive, and professional language
- Being considerate of differing opinions and experiences
- Providing constructive feedback and thoughtful code reviews
- Gracefully accepting criticism and suggestions
- Collaborating in good faith to improve the project
- Helping newcomers and encouraging learning

### Unacceptable behavior includes:
- Harassment, discrimination, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing private or sensitive information without explicit consent
- Intimidation, threats, or abusive behavior
- Any conduct that would be considered inappropriate in a professional or collaborative environment

---

## Maintainer Responsibilities
Project maintainers are responsible for:
- Enforcing this Code of Conduct fairly and consistently
- Reviewing issues, pull requests, and discussions respectfully
- Taking appropriate action when unacceptable behavior occurs
- Ensuring that community interactions remain constructive and welcoming
- Clean commits and code changes are a must in this project
- Make sure your PR is clean and is up to date

Maintainers reserve the right to remove, edit, or reject contributions that violate this Code of Conduct.

---

## Scope
This Code of Conduct applies to:
- All project spaces, including GitHub repositories, issues, pull requests, and discussions
- All interactions related to ResumeR, whether online or offline
- Any community spaces officially associated with this project

---

## Enforcement
If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers.

Reports will be reviewed promptly and handled with discretion and confidentiality.  
Depending on severity, actions may include warnings, temporary restrictions, or permanent removal from the project.
